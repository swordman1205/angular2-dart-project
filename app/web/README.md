# user_web

 * based on [Angular](https://webdev.dartlang.org/angular/)

## Tricks

 * Run locally
    1. Run
        ```shell
        cd user_web
        pub get
        pub serve
        ```
    2. Then open: [http://localhost:8080](http://localhost:8080) 
 * Run tests locally
    ```shell
    cd user_web
    pub run test
    ```
 * Run browser tests locally in chrome
    ```shell
    cd user_web
    pub serve
    pub run test --pub-serve=8081 -p chrome
    ```
